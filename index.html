I CAN DO

웹 디자인&퍼블리싱,영상편집을 할 수 있습니다!

짧은 시간동안 어설프게 많은 툴을 배웠습니다. 물리적인 시간부족이란 저만의 핑계로, 포트폴리오에 거짓으로 기재하고 싶지 않았습니다.
아직 부족한 점도 많고 제가 알지 못하는 영역 또한 무수히 많을 것입니다. 하지만 제가 다루는 도구에서 만큼은 최대로 활용하고 잘 할 자신이 있습니다.

Web Publish

웹표준을 준수하며 HTML5 마크업부터 여러 프로퍼티를 이용하여 응용할 수 있습니다. 또한 Javascript와 Jquery의 메서드를 이용해 역동적인 웹브라우저를 만들 수 있습니다.

MEDIA

PremierPro와 AfterEffect를 연동하여 사용 가능하고 최신 트렌드에 맞게 폰트,톤앤무드 등을 꾸준히 연구하고 학습하고 있습니다.

USE TOOL

Web Design이나 Media에 많이 사용될 Img파일은 편집과 보정이 원활하게 가능하며 Media자료는 컷편집,이펙트 효과,자막 활용 등 모든 툴과 연계하여 가능합니다.

EDITOR

웹 에디터로는 주로 visual studio codes를 이용하며 매니지먼트는 github으로 통합관리하고 있습니다. 작업중에도 파일 업로드와 꾸준히 Readme파일 업데이트를 실시하고 있습니다.
